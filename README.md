# 🚀 제미나이 차트 v12.0 업데이트 내용 요약

사용자님의 추가 요청을 반영하여 **클릭한 차트(활성 차트)의 배경에 실시간 코인 가격을 워터마크 형태로 크게 표시**하는 V12 업데이트를 완료했습니다.

## 🛠 주요 업데이트 내용 (V12)
1. **차트 내 실시간 가격 오버레이(워터마크) 표시**
   - 차트를 클릭하여 선택(활성화)하면, 해당 차트 중앙 배경에 코인 심볼과 현재 가격이 큰 글씨(워터마크)로 나타납니다.
   - 가격이 양봉(상승)일 때는 붉은색 계열, 음봉(하락)일 때는 푸른색 계열로 은은하게 표시되어 차트 캔들을 가리지 않으면서도 현재 가격을 한눈에 확인할 수 있습니다.
   - 실시간으로 가격이 변동될 때마다 워터마크 가격도 함께 즉시 갱신됩니다.

# 🚀 제미나이 차트 v11.0 업데이트 내용 요약

## 🛠 주요 업데이트 내용 (V11)
1. **8분할 차트 기본 설정**
   - 웹페이지(차트 앱) 최초 로딩 시, 기본 레이아웃이 기존 1분할에서 **8분할 (4x2)** 형태로 변경되었습니다.
   - 8개의 차트 창에 시가총액/인기 코인 8종(BTC, ETH, SOL, XRP, DOGE, ADA, BNB, AVAX)이 기본 배치되어 시장 전체 상황을 한눈에 파악할 수 있습니다.
2. **실시간 자동 분석 디폴트화 (3분 주기)**
   - 앱 실행 시 '실시간 자동 분석' 체크박스가 **기본으로 활성화**되도록 수정했습니다.
   - 기존 5분 주기였던 자동 갱신 주기를 더 빠르고 즉각적인 **3분 단위**로 단축했습니다.
3. **정밀 분석 수동 갱신 업그레이드**
   - 하단의 `⚡ 정밀 분석 수동 갱신` 버튼 클릭 시 즉각적인 텍스트 변경만이 아닌, 시각적인 로딩 상태(⏳ *정밀 분석 중... AI 가동*)와 지연 애니메이션을 적용했습니다. 
   - 사용자가 "분석 알고리즘이 심층 데이터를 수집하고 있다"는 직관적이고 고급스러운 느낌을 받을 수 있도록 UX를 업그레이드했습니다.

# 🚀 제미나이 차트 v10.0 업데이트 내용 요약

## 🛠 주요 업데이트 내용 (V10)
1. **차트 보조지표 및 라인 굵기 강화**
   - 3가지 이동평균선(MA 20, 60, 120)의 선 굵기를 기존보다 더 두껍게(굵게) 변경하여 추세 방향을 한눈에 명확하게 파악할 수 있도록 개선했습니다.
   - 퀀트 분석에서 제공하는 **지지선(S1, S2, S3)**과 **저항선(R1, R2, R3)**, 그리고 피벗(P) 라인 역시 기존 얇은 점선에서 두꺼운 실선(진한 색상)으로 변경하여 가격 매물대와 돌파 구간을 쉽게 식별할 수 있도록 가독성을 높였습니다.

# 🚀 제미나이 차트 v9.0 업데이트 내용 요약

## 🛠 주요 업데이트 내용 (V9)
1. **코인 검색 기능 추가**
   - 차트 상단 메뉴에 코인 검색창(🔍)을 추가했습니다. 종목 코드(예: BTC, ETH 등)를 입력하고 엔터를 누르면 현재 선택된 차트에 즉시 반영됩니다.
2. **즐겨찾기(⭐) 기능 추가**
   - 자주 보는 코인들을 '즐겨찾기' 목록에 추가할 수 있습니다. 브라우저 로컬 저장소(localStorage)를 활용하여 페이지를 새로고침하거나 나중에 다시 접속해도 저장된 목록이 그대로 유지됩니다. 드롭다운 메뉴에서 코인을 선택하면 차트에 바로 불러옵니다.
3. **선택한 차트 클릭 시 퀀트 자동 분석 연동**
   - 여러 개의 차트를 띄워놓고 있을 때, 특정 차트의 빈 공간을 **클릭**하면 해당 차트 테두리에 골드(Gold) 색상의 하이라이트가 표시됩니다.
   - 클릭과 동시에 좌측 **🤖 퀀트 분석 센터** 패널이 해당 코인의 데이터를 실시간으로 가져와 지지/저항선, 추세, RSI, 포지션 추천 등의 정밀 분석을 자동으로 업데이트합니다.

# 🚀 제미나이 차트 v7.0 업데이트 내용 요약 분할 화면에서 **개별 차트 확대/축소** 기능을 새롭게 추가한 V7 업데이트를 완료했습니다.

## 📊 주요 변경 사항

### 1. 좌측 퀀트 분석 센터 (Left AI Panel)
* 기존 우측 슬라이드 패널 방식에서 스크린샷과 동일한 **좌측 고정형 사이드바 레이아웃**으로 전면 개편했습니다.
* 시선 이동이 자연스럽게 좌측(종합 분석) ➡️ 우측(개별 차트)으로 이어지도록 설계되었습니다.

### 2. 개별 차트 확대/축소 (Expand Toggle)
* 분할 화면(2~8분할) 사용 중 특정 차트를 자세히 보고 싶을 때, 차트 우측 상단의 **`⛶` 버튼**을 클릭하면 해당 차트가 전체 화면 크기로 즉시 확대됩니다.
* 다시 누르면 원래 분할 그리드로 돌아갑니다.

### 3. 다중 차트 8분할 지원 (Multi-chart Grid)
* 상단바에서 레이아웃(1x1 ~ 4x2)을 선택하여 **최대 8개의 차트를 동시에** 모니터링할 수 있도록 구조를 전면 리팩토링했습니다.
* 각 차트마다 독립적인 코인 심볼 검색과 타임프레임(1m ~ 1M) 설정이 가능합니다.
* 바이낸스 실시간 웹소켓이 각 차트별로 개별 연동됩니다.

### 2. 3차 지지/저항선 (Level 3 S/R) 추가
* 피벗 포인트 기반의 기존 1, 2차 라인에 이어 **R3(3차 저항) 및 S3(3차 지지)** 라인을 차트에 추가했습니다.
* 보다 정밀한 단기 트레이딩 셋업을 구성할 수 있습니다.

### 3. 내장형 알고리즘 분석 (Claude API 제거)
* 지속적인 API 토큰 과금 문제를 방지하기 위해 **Claude API 연동 로직을 제거**했습니다.
* 대신, 자바스크립트에 내장된 알고리즘이 메인 차트(1번)의 데이터를 실시간으로 읽어 추세(이동평균 배열), 과매수/과매도(RSI), 돌파/이탈(S/R 라인 기준) 여부를 종합 판단하여 **한국어 브리핑을 자체 생성**합니다.
* **[실시간 자동 분석]** 토글을 켜면 5분마다 API 비용 없이 자동으로 차트 분석 리포트가 갱신됩니다.
